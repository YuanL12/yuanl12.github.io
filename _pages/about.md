I am a Master student in the [Committee on Computational and Applied Mathematics](https://cam.uchicago.edu/) at the University of Chicago. I am interested in Topological Data Analysis(TDA), Scientific Computation and Geometric Machine Learning.

## Recent work
- New pre-print released [Topology-Preserving Dimensionality Reduction via Interleaving Optimization](https://arxiv.org/abs/2201.13012) with [repository](https://github.com/CompTop/Interleaving-DR). 
- New [torch-TDA](https://github.com/CompTop/torch-tda) package released! It will do automatic differentiation for topological data analysis.
- Using Transformer (with a Encoder-Decoder structure) to decrypt encrypted message generated by RSA algorithm ([source code](https://github.com/YuanL12/DL-Pytorch/blob/gh-pages/Pytorch/Transformer%20and%20RSA.ipynb)), which shows that mordern deep learning model is a powerful tool even for traditionally unsolvable prime problem. 
- New pre-print released [Accelerating Iterated Persistent Homology Computations with Warm Starts](https://arxiv.org/abs/2108.05022) with [repository](https://github.com/YuanL12/TDA_Updating_Persistence). It also includes an interesting Pytorch layer which can increase the length of the 1st order persistent homology([source code](https://github.com/YuanL12/TDA_Updating_Persistence/blob/main/develop/BATS_hole.ipynb)), which purses more cycles.
