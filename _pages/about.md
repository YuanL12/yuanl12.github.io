I am a Master student in the [Committee on Computational and Applied Mathematics](https://cam.uchicago.edu/) at the University of Chicago. I am interested in topological data analysis(TDA), scientific computation and machine learning.

## Recent work 
- Using Transformer (with a Encoder-Decoder structure) to decrypt encrypted message generated by RSA algorithm ([source code](https://github.com/YuanL12/DL-Pytorch/blob/gh-pages/Pytorch/Transformer%20and%20RSA.ipynb)), which shows that mordern deep learning model is a killer even for traditionally unsolvable prime problem. 
- [Accelerating Iterated Persistent Homology Computations with Warm Starts](https://arxiv.org/abs/2108.05022) with github [repository](https://github.com/YuanL12/TDA_Updating_Persistence). We develop an approach to accelerate persistent homology computations performed on many similar filtered topological spaces which is based on updating associated matrix factorizations. It also includes an interesting Pytorch layer which can increase the length of 1st order persistent homology([source code](https://github.com/YuanL12/TDA_Updating_Persistence/blob/main/develop/BATS_hole.ipynb)), which forms more and more cycles. Furthermore, we will have a Torch-TDA package coming soon!
